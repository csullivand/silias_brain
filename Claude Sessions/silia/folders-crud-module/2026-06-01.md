## Decisions Made
- Folders are per-account, not per-chatbot (cross-domain: agents, tables, dashboards)
- Use existing requireRole() auth — swap to can() when Feature 0 lands
- Denormalized depth + path fields to avoid recursive DynamoDB queries
- DynamoDB TTL for 30-day auto-purge of soft-deleted folders
- Async SQS for bulk operations with >50 descendants
- Reuse BillingAuditLog table for folder audit entries
