### 4. Workflows CORS = actually 401 on OPTIONS preflight (NOT the CASL grant)
- `GET /v1/workflows` browser 'CORS' error. Real cause: OPTIONS preflight returns **401 UnauthorizedException**. API sets `DefaultAuthorizer` but MISSING `AddDefaultAuthorizerToCorsPreflight: false` → SAM attaches authorizer to OPTIONS → preflight (no token) → 401 → DEFAULT_4XX has no ACAO → browser says CORS.
- **Fixed:** added `AddDefaultAuthorizerToCorsPreflight: false` to Auth block in **Workflows** + **Integrations** templates. (Metrics false-positive: comment only, no real DefaultAuthorizer.)
